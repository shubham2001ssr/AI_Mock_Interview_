# InterviewGenie Knowledgebase

## Overview
A concise reference for interview topics, question types, and curated learning resources used by InterviewGenie. It supports both business (MBA) and technical (CS) interview tracks.

## Interview Tracks
- **Business Track**: Finance, Operations, Consulting, Marketing, Strategy
- **Technical Track**: Data Structures & Algorithms, System Design, Programming Languages (Java, Python, JavaScript, C++), DevOps, Cloud, Databases

## Question Types
| Type | Description |
|------|-------------|
| **Technical Coding** | Coding problems with focus on algorithmic concepts and complexity analysis |
| **System Design** | High‑level architecture scenarios (scalability, reliability, trade‑offs) |
| **Technical HR** | Behavioral questions specific to engineering roles |
| **Case Study** | Business scenarios requiring market sizing, profitability analysis |
| **Behavioral** | STAR‑method questions on leadership, teamwork, conflict resolution |
| **Finance/Operations** | Quantitative and analytical questions for MBA roles |

## Core Resources
- **LeetCode** – Practice coding problems (https://leetcode.com)
- **GeeksforGeeks** – Algorithms and data‑structures tutorials (https://geeksforgeeks.org)
- **System Design Primer** – System design concepts (https://github.com/donnemartin/system-design-primer)
- **Coursera / edX** – Courses for Java, Python, Cloud, and Business topics
- **Books** – *Introduction to Algorithms* (CLRS), *Case in Point* (consulting), *Financial Modeling* (Finance)

## How to Use
1. Select a **track** (Business or Technical) when creating an interview.
2. Choose **focus areas** (e.g., "algorithms", "valuation").
3. InterviewGenie generates tailored questions based on the selected type.
4. After feedback, the **Learning Resources** component surfaces relevant items from the knowledgebase.
